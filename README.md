
### 环境变量

- `.env.dev`  本地
- `.env.test` 测试
- `.env.pre`  预发布 
- `.env.prod` 线上

**启动服务**
`npm run dev`

**测试打包**
`npm run build:test`

**预发布打包**
`npm run build:pre`

**线上打包**
`npm run build:prod`
